# PredictIA

## Objetivo

Crear una plataforma de predicciones donde los usuarios participen mediante WhatsApp respondiendo encuestas sobre distintos eventos.

La plataforma deberá:

- Registrar usuarios.
- Enviar encuestas por WhatsApp.
- Recibir y almacenar las respuestas.
- Generar una predicción mediante IA.
- Comparar la predicción de la IA con la opinión de los usuarios.
- Mostrar estadísticas e historial en una página web.
- Incorporar suscripciones mediante Mercado Pago en una etapa futura.
